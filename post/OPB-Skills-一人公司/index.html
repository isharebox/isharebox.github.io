Background

In the AI era, One Person Business (OPB) becomes possible. A single person can accomplish tasks that previously required an entire team with the help of AI.

However, general-purpose AI assistants often lack deep professional knowledge and cannot work as efficiently as real experts. OPB-Skills solves this problem ‚Äî it's a collection of professional Skills designed for Claude Code, where each Skill is a "virtual expert" equipped with domain-specific knowledge, workflows, and best practices.

Core Philosophy: Enable solo entrepreneurs to have their own "virtual expert team" covering all aspects of business operations including product, R&D, design, marketing, operations, finance, and legal.

Core Value

Value Description Professional Depth Each Skill comes with built-in domain expertise, methodologies, and templates ‚Äî not generic responses Workflow-Driven Provides structured processes from input to output Plug and Play Based on Claude Code Skill mechanism, ready to use after installation Continuous Evolution Iteratively improved based on actual usage, accumulating best practices

Quick Start

Prerequisites

  • Claude Code CLI installed

  • Claude Pro/Team subscription

Installation

Method 1: Global Installation (Recommended)

bash
# Clone the repository
git clone https://github.com/chendongqi/OPB-Skills.git

# Link skills directory to Claude Code global config
ln -s /path/to/OPB-Skills/skills ~/.claude/skills

Method 2: Project-Level Installation

bash
# Copy to project .claude directory
cp -r /path/to/OPB-Skills/skills /your-project/.claude/

Contributing

How to Contribute New Skills

  1. Use god-skill-creator to create Skill skeleton

  3. Test and validate, then submit PR

Skill Design Principles

  • Simplicity First: Only include professional knowledge that Claude doesn't inherently have

  • Workflow-Driven: Provide structured execution steps

  • Progressive Loading: Core content in SKILL.md, detailed references in references/
